
 ProyectoFinalPA — Tienda Agrícola (Java + GUI)

Descripción  
Proyecto de tienda agrícola con clases para Productos (Semilla, Fertilizante, Insecticida), manejo de carrito de compras, árbol binario de precios, lista enlazada para carrito, y una interfaz gráfica con JFrame. Permite cargar productos, buscarlos, agregarlos al carrito, calcular totales, hacer checkout y generar factura.

 Características principales  
- Clases para diferentes tipos de productos (Semilla, Fertilizante, Insecticida)  
- Árbol binario eficiente para manejo y búsqueda por precio  
- Lista enlazada para carrito de compras  
- Interfaz gráfica (Java Swing) para interacción con el usuario  
- Carrito funcional: agregar productos, ver totales, checkout y factura  
- Código comentado y organizado  

Tecnologías usadas  
- Java (versión X)  
- Swing (JFrame, paneles)
